# 夹爪力位混合控制设计文档

## 概述

基于达妙 DM-J4310-2EC 电机的三模式夹爪控制器,使用 MIT 阻抗控制模式实现:

1. **纯位置控制** — 无视外力,强制移动到目标位置
2. **纯力矩控制** — 不管位置,始终输出恒定力矩
3. **力位混合控制** — 向目标运动,遇力则停/退,力消则继续

---

## 控制理论

### MIT 阻抗控制方程

电机内部以 10kHz+ 的 FOC 频率执行:

```
τ = Kp × (q_des - q) + Kd × (dq_des - dq) + τ_ff
```

| 参数 | 范围 | 说明 |
|------|------|------|
| Kp | 0-500 | 位置刚度增益 |
| Kd | 0-5 | 速度阻尼增益 |
| q_des | ±12.5 rad | 目标位置 |
| dq_des | ±30 rad/s | 目标速度 |
| τ_ff | ±10 Nm | 前馈力矩 |

**重要**: Kp ≠ 0 时必须 Kd ≠ 0,否则会振荡。

### 三种模式的 MIT 参数配置

| 模式 | Kp | Kd | q_des | dq_des | τ_ff |
|------|----|----|-------|--------|------|
| 纯位置 | 20 (高刚度) | 2.0 | 目标位置 | 0 | 0 |
| 纯力矩 | 0 | 0.5 (抑制振动) | 0 | 0 | 目标力矩 |
| 混合-跟踪 | 20 | 2.0 | 目标位置 | 0 | 0 |
| 混合-保持 | 3 (低刚度) | 1.0 | 接触位置 | 0 | 0 |
| 混合-柔顺 | 0 | 1.5 | 0 | 0 | 0 |

## 力位混合状态机

```
                    ┌────────────────────────────────────────┐
                    │                                        │
                    v                                        │
    ┌───────────┐  |τ| ≥ limit   ┌───────────┐              │
    │ TRACKING  │ ─────────────> │  HOLDING  │              │
    │ Kp=高     │                │  Kp=低    │              │
    │ q=目标    │                │  q=冻结   │              │
    └───────────┘ <───────────── └───────────┘              │
          │       |τ| < 80%limit    │                       │
          │       (迟滞防振荡)       │ |τ| ≥ 150%limit      │
          │                         v                       │
          │                    ┌───────────┐                │
          │                    │ COMPLYING │                │
          └──────────────────  │  Kp=0     │ ───────────────┘
            |τ| < 80%limit     │  Kd=阻尼  │   |τ| < limit
                               └───────────┘
```

### 状态转移说明

- **TRACKING → HOLDING**: 反馈力矩达到阈值,冻结当前位置,切换到低刚度保持
- **HOLDING → TRACKING**: 力矩降到阈值的 80% 以下(迟滞),恢复向目标运动
- **HOLDING → COMPLYING**: 外力持续增大超过阈值的 150%,释放位置约束,纯阻尼跟随
- **COMPLYING → TRACKING**: 外力减小到阈值 80% 以下,恢复位置跟踪

迟滞比 (默认 0.8) 防止在阈值附近反复切换导致振荡。

## 安全机制

| 保护项 | 阈值 | 动作 |
|--------|------|------|
| MOS 管温度 | > 80°C | 立即禁用电机 |
| 转子温度 | > 100°C | 立即禁用电机 |
| 电机错误码 | ≠ 0 或 1 | 立即禁用电机 |
| 反馈超时 | > 50ms 无反馈 | 立即禁用电机 |
| 力矩钳位 | ±8 Nm (可配置) | 软件限幅,防超扭 |
| 位置钳位 | pos_open ~ pos_close | 防超行程 |
| 启动斜坡 | 100ms Kp=0 | 防使能瞬间冲击 |

## 参数调节指南

### 初始调试步骤

1. 设 Kp=0, 逐渐增大 Kd (0.5 → 2.0),手转电机直到感觉阻尼合适
2. 缓慢增大 Kp (5 → 20 → 50),直到位置保持足够硬
3. 如果出现振荡,增大 Kd 或减小 Kp;Kp/Kd 比值建议 < 20

### DM4310 推荐参数

| 场景 | Kp | Kd |
|------|----|----|
| 柔顺保持 (易碎物) | 3-8 | 1.0-2.0 |
| 正常夹持 | 10-20 | 1.5-2.5 |
| 刚性定位 | 30-50 | 2.0-3.0 |
| 力矩模式阻尼 | 0 | 0.3-1.0 |

### 力矩反馈说明

电机反馈的 τ 是基于电流估算的,不是直接力矩传感器测量值。误差约 5-15%,受摩擦、齿槽效应和温度漂移影响。低于 0.5 Nm 的精细力控可能需要齿槽补偿。

## API 使用示例

```python
from dm_motor import DmDevice, GripperController

dev = DmDevice()
dev.open(nom_baud_hz=1_000_000, dat_baud_hz=5_000_000, canfd=True, brs=True)

with GripperController(dev, can_id=0x01, mst_id=0x11,
                        pos_open=0.0, pos_close=3.14) as gripper:
    # 纯位置: 闭合
    gripper.move_to(3.14)

    # 纯力矩: 施加 2 Nm
    gripper.set_torque(2.0)

    # 力位混合: 闭合到 3.14 rad,力限 3 Nm
    gripper.grip(3.14, force_limit=3.0)

    # 便捷方法
    gripper.close_gripper(force_limit=2.0)
    gripper.open_gripper()

    # 读状态
    state = gripper.get_state()
    print(state)

dev.close()
```
